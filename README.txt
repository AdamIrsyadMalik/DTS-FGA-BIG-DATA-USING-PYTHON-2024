Sistem Inventory Management Stok Barang

Deskripsi

Sistem Inventory Management Stok Barang adalah aplikasi yang digunakan untuk mengelola stok barang di gudang. Aplikasi ini dapat digunakan untuk menambah, mengupdate, menghapus, dan menampilkan data stok barang.

Fitur

    Menambah data stok barang
    Mengupdate data stok barang
    Menghapus data stok barang
    Menampilkan data stok barang
    Menampilkan grafik stok barang berdasarkan jumlah
    Menampilkan 10 barang dengan stok tertinggi
    Menampilkan 10 barang dengan stok terendah
    Menampilkan 10 barang dengan harga tertinggi
    Menampilkan 10 barang dengan harga terendah

Cara Menggunakan

    Jalankan aplikasi dengan membuka file main.py
    Masukkan data stok barang dengan mengklik tombol "Tambah"
    Update data stok barang dengan mengklik tombol "Update"
    Hapus data stok barang dengan mengklik tombol "Hapus"
    Tampilkan data stok barang dengan mengklik tombol "Tampilkan"
    Tampilkan grafik stok barang berdasarkan jumlah dengan mengklik tombol "Grafik Jumlah"
    Tampilkan 10 barang dengan stok tertinggi dengan mengklik tombol "Top 10 Stok Tertinggi"
    Tampilkan 10 barang dengan stok terendah dengan mengklik tombol "Top 10 Stok Terendah"
    Tampilkan 10 barang dengan harga tertinggi dengan mengklik tombol "Top 10 Harga Tertinggi"
    Tampilkan 10 barang dengan harga terendah dengan mengklik tombol "Top 10 Harga Terendah"

Persyaratan

    Python
    MySQL
    matplotlib
    tkinter

Instalasi

    Instal Python
    Instal MySQL
    Instal matplotlib dengan menjalankan perintah pip install matplotlib
    Instal tkinter dengan menjalankan perintah pip install tk
    Jalankan aplikasi dengan membuka file main.py
